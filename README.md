# Inventario de Tienda en C#

Este es un sistema de gestión de inventario básico desarrollado en consola utilizando **C#** y características modernas de .NET.

## 🚀 Características
- **Registro de Productos:** Permite añadir nuevos artículos validando que los datos no estén vacíos.
- **Validación de Datos:** Control de excepciones con bloques `try-catch` para evitar cierres inesperados al ingresar números o precios.
- **Formateo Visual:** Consulta de inventario organizada en columnas alineadas de forma limpia en la consola.

## 🛠️ Conceptos de Programación Aplicados
- Programación Orientada a Objetos (Clases, Propiedades y Constructores).
- Estructuras de control y ciclos (`while`, `switch`, `if-else`).
- Colecciones dinámicas (`List<T>`).
